Installation:

Very Easy to install "Newsletter Subscribe" extension

1). Upload all extension files in your root folder of your Store Server.

2). Now go to Extension > Installer, and Install the "newsletter_subscribe.ocmod" file,which is available in ocmod folder.

3). After Install the ocmod file then goto the Extension > Modification menu and refresh the modification.

4). After that go to extension > module. Under Module find "Newsletter Subscribe" extension and Click on install button.

5). After previous step click on edit button and step up the extension.

6). It's Done and have a nice day
